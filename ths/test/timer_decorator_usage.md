# timer 装饰器使用说明

## 功能说明

`timer` 是一个用于测量方法执行耗时的装饰器，可以精确到秒级（带小数点）。

## 使用方法

### 1. 导入装饰器

装饰器已经定义在 `ths/trader.py` 文件中，无需额外导入。

### 2. 应用装饰器

在需要测量耗时的方法上添加 `@timer` 装饰器即可：

```python
@timer
def your_method(self, *args, **kwargs):
    # 方法实现
    pass
```

### 3. 示例

例如，在 `ThsTrader` 类的 `get_account_summary` 方法上应用装饰器：

```python
@timer
def get_account_summary(self) -> Dict[str, Any]:
    """
    获取账户概览信息
    返回: 总资产、浮动盈亏、总市值、当日参考盈亏
    """
    # 方法实现
    pass
```

## 输出效果

当调用带有 `@timer` 装饰器的方法时，会在日志中输出类似以下的信息：

```
2026-03-14 15:30:45,678 - ths.trader - INFO - ⌛ 方法 get_account_summary 执行耗时: 2.345 秒
```

## 注意事项

1. 装饰器使用了 `logging` 模块记录耗时信息，请确保日志级别设置为 `INFO` 或更低。

2. 装饰器会捕获方法执行过程中的异常，但仍会将异常抛出，以便调用者处理。

3. 装饰器使用了 `functools.wraps` 保留了原方法的元信息（如函数名、文档字符串等）。

## 扩展使用

如果需要在其他类或模块中使用此装饰器，可以将装饰器代码复制到相应的文件中，或创建一个单独的工具模块来存放装饰器。
