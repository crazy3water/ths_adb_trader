# 架构设计文档

## 系统概述

Watery Traders 是一个基于 uiautomator2 的安卓手机自动化交易系统，通过模拟人工操作同花顺APP实现自动化交易功能。

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                        客户端                                │
│  (curl / Postman / Claude Desktop / 浏览器)                  │
└────────────────────────┬────────────────────────────────────┘
                         │ HTTP / MCP
┌────────────────────────▼────────────────────────────────────┐
│                    FastAPI Server                            │
│                      (:8000)                                 │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────┐  ┌──────────────────┐    │
│  │  设备管理   │  │  交易操作   │  │   MCP Server     │    │
│  │  /device/*  │  │   /ths/*    │  │   (可选 :19090)   │    │
│  └─────────────────┘ ─┘  └──────── └──────────────────┘    │
└────────────────────────┬────────────────────────────────────┘
                        │ uiautomator2
┌────────────────────────▼────────────────────────────────────┐
│                    Android Device                            │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                   同花顺 APP                          │   │
│  │   (com.hexin.plat.android)                          │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

## 核心模块

### 1. config.py - 配置模块

负责系统配置管理：
- `DEVICE_SERIAL`: 手机连接地址（IP:端口）
- `THS_PACKAGE`: 同花顺APP包名
- `SCREENSHOT_DIR`: 截图保存目录

### 2. trader.py - 交易核心模块

`ThsTrader` 类是系统的核心，包含所有交易逻辑：

#### 设备连接
```python
def connect(self) -> bool
```

#### 基础操作
```python
def click_by_text(text: str) -> bool      # 文字点击
def input_text(text: str) -> bool           # 输入文本
```

#### 交易操作
```python
def buy(code, price, quantity, mode) -> Dict   # 买入
def sell(code, price, quantity, mode) -> Dict   # 卖出
def cancel_order(index: int) -> Dict        # 撤单
```

#### 账户查询
```python
def get_account_summary() -> Dict            # 获取账户概览
def get_positions() -> Dict                  # 获取持仓列表
```

### 3. app.py - API服务模块

基于 FastAPI 实现的 RESTful API 服务，提供完整的 HTTP 接口。

### 4. mcp_server.py - MCP模块（可选）

基于 Model Context Protocol 的服务器，支持与 Claude Desktop 集成。

## 交易流程

### 买入流程

```
进入交易页面 → 点击"买入" → 输入股票代码 → 输入价格 → 输入数量 → 确认 → 截图 → 返回
```

### 撤单流程

```
进入交易页面 → 点击"撤单" → 选择委托 → 确认撤单 → 返回
```

## 扩展性

### 添加新功能

1. 在 `trader.py` 中添加新方法
2. 在 `app.py` 中添加对应的路由
3. 如需MCP支持，在 `mcp_server.py` 中添加工具
